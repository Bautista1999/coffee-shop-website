# Coffee Shop Website

A modern, responsive website for a coffee shop featuring:

- Clean, modern design
- Mobile-responsive layout
- Optimized performance
- Easy content management
- Fast loading times

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript
- Vercel deployment

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel with automatic deployments from the main branch.
